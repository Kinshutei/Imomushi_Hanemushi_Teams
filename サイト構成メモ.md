## データ

| 項目 | 内容 |
|------|------|
| ファイル名 | streaming_info.csv |
| 管理場所 | リポジトリルート |
| 取得方法 | GitHub Raw URL から fetch |
| 更新タイミング | 手動（歌枠終了後） |

### CSVの主なカラム

- 枠名
- 楽曲名
- 歌唱順
- 配信日
- 枠URL
- コラボ相手様
- 原曲Artist
- 作詞
- 作曲
- リリース日

---

## タブ構成

### LiveStreaming Info
- 配信枠ごとのセットリスト表示
- アコーディオン形式（アニメーション付き開閉）
- YouTube サムネイル表示
- 曲名検索機能
- 全て開く／全て閉じるボタン

### Uta-Mita DB
- 楽曲ごとの集計テーブル
- 列ヘッダークリックでソート
- 歌唱回数ランキング棒グラフ（上位20曲）
- リリース年度分布ツリーマップ
- 原曲アーティスト分布ツリーマップ

### About
- サイト説明
- 構築目的
- タブの使い方
- データについて
- リンク（YouTube・X）
- 免責事項
